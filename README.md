# business-statistics-arhangelsk
Комплексный экономический анализ Архангельской области (2017–2024)  

 Что внутри
- Статистический анализ инвестиций, основных фондов, рынка труда и демографии
- Расчёт 12+ показателей: темпы роста, коэффициенты опережения, фондоотдача
- Факторный анализ прироста ВРП
- Корреляционный анализ (r = 0,969)
- Прогнозирование методом МНК
- Выводы
